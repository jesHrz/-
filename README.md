# SDU选课助手
主要功能查询课程和监视课余量
通过“python run.py"运行程序或直接执行release目录下的run.exe

刷新间隔的单位为毫秒(ms), 不要输入非法内容

开课院校里请输入院校代码, 例如29(体育学院), 13(计算机学院)

一个人码代码不仔细，有bug的话还请大家即时通知我哦
