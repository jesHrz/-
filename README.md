# SDU选课助手
主要功能查询课程和监视课余量
通过“python run.py"运行程序或直接执行release目录下的exe文件

刷新间隔的单位为毫秒(ms)

可以在课程名里输入课程类型或课程名筛选课程，教师名和开课学院用法与课程名一样

上课星期、上课节次和上课地点三个下拉框中空白表示所有，默认选中
